# Lab 07 – Biomechaniczna roślina w Blender GUI z zaawansowanym Edit Modem

## Co zostało zrealizowane
Przy pomocy interfejsu graficznego Blendera 5.1 utworzona została scena 3D przedstawiająca biomechaniczną roślinę składającą się z elementów modelowanych w Edit Mode z zastosowanymi modyfikatorami niedestrukcyjnymi (Mirror, Subdivision, Screw, Shrinkwrap).

## Uruchomienie
Pobrać plik `roslin.blend` i otworzyć go w Blenderze.

## Uwagi
Wszystkie wykorzystane tekstury są w domenie publicznej (CC0) i pochodzą z Poly Haven.
